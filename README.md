# Modern Art Gallery

Sitio web de una galería de arte moderna construido con Vite, HTML y Sass.

## 🚀 Instalación

1. Clona el repositorio:
```bash
git clone <url-del-repositorio>
cd sass-modern-art-gallery
```

2. Instala las dependencias:
```bash
npm install
```

## 📝 Scripts Disponibles

### Desarrollo
Inicia el servidor de desarrollo con hot-reload:
```bash
npm run dev
```
El sitio estará disponible en `http://localhost:5173`

### Build
Genera la versión de producción:
```bash
npm run build
```

### Preview
Previsualiza la versión de producción:
```bash
npm run preview
```

## 🛠️ Tecnologías

- **Vite** - Build tool y servidor de desarrollo
- **Sass** - Preprocesador CSS
- **HTML** - Estructura del sitio

## 📁 Estructura del Proyecto

```
sass-modern-art-gallery/
├── src/
│   ├── assets/        # Imágenes e iconos
│   ├── pages/         # Páginas HTML
│   ├── styles/        # Archivos Sass
│   └── main.js        # Punto de entrada
├── index.html         # Página principal
└── package.json       # Configuración del proyecto
```

## 📄 Páginas

- **Home** (`index.html`) - Página principal con galería
- **Location** (`src/pages/location.html`) - Página de ubicación

## 🎨 Diseño Responsive

El sitio está optimizado para:
- **Desktop** - Ancho completo
- **Tablet** - ≤ 768px
- **Mobile** - ≤ 480px
